Acceso al proyecto de Super Mario Bros

Pasos para ingresar correctamente:

  1. Instale y Configure XAMPP y ejecute Apache y MySQL.
  2. Arrastre la carpeta Super-Mario-Bros a la siguiente dirección: C:\xampp\htdocs u a la carpeta htdocs dependiendo de en donde haya instalado XAMPP.
  3. Ingrese a phpMyAdmin y cree lo siguiente:
    • Una Base de Datos que se llame: registro
    • Dentro de "registro" una Tabla que se llame: datosdeusuario
  4. Una vez hecho los pasos anteriores, podrá accesar a mi proyecto.
